# Freitag Scroll

> 계산식 scroll 이벤트를 활용한 업사이클링 브랜드 프라이탁의 회사소개 리뉴얼

<br>

[🖥️ 포트폴리오 보러가기](https://yongzin.github.io/scroll/scroll.html)

<br>

> ### 미리 보기
> ![preview](https://github.com/yongZin/yongZin/blob/main/images/gif/scroll.gif?raw=true)

> ### 주요 기능
> - canvas 활용
> - scroll에 반응하는 background-image 와 텍스트
> - IntersectionObserver를 사용하지 않고 높이 계산을 통한 scroll 이벤트
> - 텍스트의 가독성을 높이기 위한 미니멀한 반응형 UI
